# 📘 구구단 (백준 2739번)

📅 데일리 알고리즘 연습 | 2025.07.24  
🔗 [문제 링크](https://www.acmicpc.net/problem/2739)

---

## 📌 문제 설명

N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오.  
출력 형식에 맞춰서 출력하면 된다.

---

## ✍️ 입력

- 첫째 줄에 N이 주어진다.  
- (1 ≤ N ≤ 9)

---

## ✅ 출력

- N * 1부터 N * 9까지 한 줄에 하나씩 출력한다.  
- 출력 형식: `N * i = 결과`
