# 📘 A + B 계산식 출력 (프로그래머스)

📅 데일리 알고리즘 연습 | 2025.07.30  
🔗 [문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/181947)

---

## 📌 문제 설명

두 정수 a, b가 주어질 때 다음과 같은 형태의 계산식을 출력하는 프로그램을 작성하세요.

출력 예:  
a + b = c

---

## ✍️ 입력

- 공백으로 구분된 두 정수 a, b가 주어집니다.  
- (1 ≤ a, b ≤ 100)

---

## ✅ 출력

- `a + b = c` 형식으로 출력합니다.  
- 예: `4 + 5 = 9`

---

## 💡 입출력 예시

| 입력 | 출력        |
|------|-------------|
| 4 5  | 4 + 5 = 9   |
| 10 2 | 10 + 2 = 12 |
